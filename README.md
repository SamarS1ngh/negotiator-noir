# Negotiator — Noir reboot (concept)

Rebooted direction (2026-07-15): a **manipulation duel** game. You read, break, scheme,
and manipulate one person at a time across a table — reading their mood, deducing their
hidden agenda, finding their personality levers — inside a noir crime story with a clear
end goal.

**Look direction: "living crime graphic-novel"** — bold, high-contrast, cinematic crime
comic art that *moves* and reacts, not static panels.
